# LibreAirspace 自由空域
### 这是什么？
这是一款开源的，类《战争雷霆》的空战游戏。旨在为《战争雷霆》玩家提供免费的下位替代。
### 游戏的运营方式是什么？
本游戏完全开源，社区共建。欢迎其他人加入。
### 本游戏的结构：
- 游戏房间服务器负责主要游戏内容。
- 房间服务器负责管理多个不同房间服务器。房间服务器只有在房间列表服务器里注册之后才能为玩家提供服务。
- 玩家在房间列表服务器中注册，登录后，请求房间列表，在GUI上显示，可以点击进入。
